# Gerendo (the SaaS)

> **Note:** "Gerendo" here means the SaaS product, not the agency identity. Same name, different workstream. Don't confuse them.

A multi-tenant SaaS that ingests an agency's tools (Gmail, Drive, Asana, Meet transcripts, WhatsApp Business, Discord) and turns the company into a single queryable brain. Think Glean, but for 15–50 person marketing agencies in EU/LatAm.
